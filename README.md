# Data-Structure

A small repository of common coding problems solved in Python, organized by data structure type.

## Contents

- `Arrays/`
  - `1-Two-Sum.py`
  - `121-Best-Time-to-Buy-and-Sell-Stock.py`
  - `283-Move-Zeroes.py`
- `LinkedList/`
  - `203-Remove-Linked-List-Elements.py`

## Description

This repository contains sample solutions for classic algorithm problems using Python. Each folder groups solutions by the primary data structure used in the problem.

## Usage

Open any `.py` file and run it with Python to see the implementation for that problem.

```bash
python Arrays/1-Two-Sum.py
```
